Introduction

Ant is a fully responsive admin web App built with Bootstrap & ReactJS. It uses Sass/Scss CSS which makes it easy to modify.

Introduction to some important files:

config-overrides.js
By default, create-react-app has no configuration, in this project, react-app-rewired is used for LESS css support
package.json
is the JSON file of NPM (You can click the link to know more). This file is used by npm to store metadata for projects published as npm modules.
package-lock.json
is for NPM users, so that others can get precisely the same dependencies.

How to Run the Demo?

Note: build folder contains the built distribution (production) version, which is for demo purpose only

You need to put the files on a server to run it. Here we'll talk about how to build a server locally.

You can build a locally server using WAMP or MAMP (if you uses PHP). Also it's pretty easy to build one with Python or Ruby (if you prefer Python or Ruby). And here we'll build a local server with NodeJS

Preparation:

You'll need to install Node.js >=v8.0.0 (NPM comes along with it) and node-static (a NPM package)


Steps:

It's easy if you are familiar with Node. Download Node.js here then install it, and install node-static package with one command line npm install -g node-static

$ npm install -g node-static

Using node-static is easy. Just enter the "build" directory with command line

$ cd build/

type static and hit Return key

$ static

It'll return something like this: 

serving "." at http://127.0.0.1:8080

Then you can view the template on http://127.0.0.1:8080 with your browser

Note: Because the build/ folder built with webpack uses absolute path, you need to run it from root path

How to Use It?

In order to do development with it, first you'll need have the following tools installed:

  • Node.js >=v8.0.0 (NPM comes along with it), just download and install it, that's it.

    • These tools are for development purpose only, you don't need them in production environment, meaning you can use any back-end languages you like, NodeJS is just one of the options, not required


    A few steps to get started:

    1. Enter "ant" folder with your command line tool.

    $ cd ant

    2. Install all the NPM packages with command: 

    $ npm install

    3. Then type npm start and you are ready to go :)

    $ npm start

    4. It will automatically open the web app with your default browser


    Note: It takes a bit long in the first time, about 20 seconds

    Note: All the source code for development are inside ./ant/src/ folder.

    Layouts & Color Schemes

    You can simply update a few variables to change the default settings.

    Open ant/src/constants/appConfig.js file, you'll find following code. You can change the value to update the default settings.

    const APPCONFIG = {
  brand: 'Ant',
  year: year,
  AutoCloseMobileNav: true,           // boolean: true, false. Automatically close sidenav on route change (Mobile only)
  showCustomizer: true,               // boolean: true, false. Customizer will be removed completely when set to false
  visibleCustomizer: false,           // boolean: true, false. Customizer will be opened (visible) when web app loaded if set to true
  color: {
    primary:  '#1890ff',
    success:  '#66BB6A',
    info:     '#01BCD4',
    infoAlt:  '#948aec',
    warning:  '#ffc53d',
    danger:   '#ff4d4f',
    text:     '#3D4051',
    gray:     '#EDF0F1'
  },
  settings: {
    layout: '1',
    boxedLayout: false,               // boolean: true, false
    fixedSidenav: true,               // boolean: true, false
    fixedHeader: true,                // boolean: true, false
    collapsedNav: false,              // boolean: true, false
    offCanvasNav: false,              // boolean: true, false
    sidenavWidth: 240,                // number
    colorOption: '34',                // string: 11,12,13,14,15,16; 21,22,23,24,25,26; 31,32,33,34,35,36
    theme: 'light',                   // (WIP) string: light, gray, dark
  }
}
